## Introduction

Ce cours de chinois mandarin couvre les bases de la communication orale — présentations, questions sur l'identité, les goûts et la vie quotidienne. Les phrases sont présentées en **pinyin** (translittération phonétique) avec leur traduction française.

## Prononciation — Pinyin

Le pinyin est le système officiel de romanisation du mandarin. Quelques règles clés :

| Son pinyin | Prononciation approx. |
|------------|----------------------|
| x | « ch » doux (entre ch et s) |
| q | « tch » léger |
| zh | « dj » |
| c | « ts » |
| z | « dz » |
| ü / u (après j, q, x) | « u » français |

## Phrases fondamentales

### Se présenter et se saluer

| Pinyin | Traduction |
|--------|-----------|
| Nǐ hǎo ! | Bonjour ! |
| Nǐ jiào shénme míngzi ? | Comment t'appelles-tu ? |
| Wǒ jiào [prénom]. | Je m'appelle [prénom]. |
| Nǐ hǎo ma ? | Comment vas-tu ? |
| Wǒ hěn hǎo. | Je vais très bien. |
| Wǒ yě hěn hǎo. | Moi aussi, je vais très bien. |
| Zàijiàn ! | Au revoir ! |

## Structures grammaticales clés

### Négation avec 不 (bù)

Pour nier un verbe, on place **bù** devant lui :
- Wǒ **bù** xǐhuān → Je **n'**aime **pas**
- Wǒ **méiyǒu** gēge → Je **n'**ai **pas** de grand frère (`méi` pour `yǒu` = avoir)

### Questions avec 吗 (ma)

Placer **ma** à la fin d'une phrase affirmative pour former une question :
- Nǐ shì xuéshēng **ma** ? → Es-tu étudiant(e) ?
- Nǐ xǐhuān Zhōngguó **ma** ? → Aimes-tu la Chine ?

### Réponse avec 也 (yě) — "aussi"

- Wǒ **yě** shì Fǎguó rén. → Moi **aussi**, je suis Français(e).

## Résumé

- La structure de base en mandarin est : **Sujet + Verbe + Complément** (sans conjugaison !).
- La négation se forme avec **bù** (不) devant le verbe, sauf pour `yǒu` (avoir) → `méiyǒu`.
- Les questions s'obtiennent en ajoutant **ma** (吗) en fin de phrase.
- Le pinyin permet d'écrire phonétiquement le mandarin — indispensable pour l'apprentissage initial.
