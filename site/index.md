## Ajouter un cours rapidement
1. Copier `site/_templates/template_cours.md`.
2. Creer le fichier dans `site/cours/SX/Nom Du Cours/`.
3. Remplir les metadonnees (`title`, `semestre`, `type_cours`, `tags`).
4. Ajouter le lien ici.
