## Ajouter un cours
<a id="ajouter-un-cours"></a>

1. Copier `site/_templates/template_cours.md`.
2. Créer le dossier `site/cours/SX/Nom-Du-Cours/` et y placer `index.md`.
3. Remplir les métadonnées (`title`, `semestre`, `type_cours`, `tags`).
4. Ajouter l'entrée dans `site/_data/navigation.yml`.
5. Commit puis push pour publier automatiquement via GitHub Actions.
