  <p class="guide-card-title"><span aria-hidden="true">📎 </span>Ajouter un cours</p>
  <ol>
    <li>Copier <code>site/_templates/template_cours.md</code>.</li>
    <li>Créer le dossier <code>site/cours/SX/Nom-Du-Cours/</code> et y placer <code>index.md</code>.</li>
    <li>Remplir les métadonnées : <code>title</code>, <code>semestre</code>, <code>type_cours</code>, <code>tags</code>.</li>
    <li>Ajouter l'entrée dans <code>site/_data/navigation.yml</code>.</li>
    <li>Commit puis push — GitHub Actions publie automatiquement.</li>
  </ol>
